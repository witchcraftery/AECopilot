API Key Configuration

The OpenAI API key is now configured via the .env file in the project root directory.

To set your API key:

  1. Create or edit the .env file in the project root
  2. Add the following line: VITE_OPENAI_API_KEY=your_actual_openai_key
  3. Replace your_actual_openai_key with your OpenAI API key
  4. Restart the extension or rebuild the project

Note: Make sure to add .env to your .gitignore file to prevent committing your API key to version control.
